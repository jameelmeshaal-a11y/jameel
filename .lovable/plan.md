

# تشخيص: جدار 150مم مسعّر بـ 176 وجدار 200مم مسعّر بـ 153

## المشكلة بالتفصيل

### بند جدار 150مم → سعر 176 (خطأ)
المحرك ربطه بسجل **"بلاطة مجوفة سمك 150 مم"** (hollow slab) بسعر 176 ريال — وهو بند مختلف تماماً! حدث هذا لأن كلاهما يحتوي "سمك 150 مم" فمرّ من Thickness Gate بنجاح.

**المكتبة تحتوي على السجل الصحيح:** `حوائط سمك 150مم` بسعر **189.75** ريال — لكن المحرك لم يجده.

### بند جدار 200مم → سعر 153.4 (خطأ)
المحرك ربطه بسجل **"بلاط تراتزو (موزايكو) مقاس 300×300×30 مم"** بسعر 153.4 ريال — بلاط أرضي لا علاقة له بالبلوك!

**المكتبة تحتوي على السجل الصحيح:** `حوائط سمك 200مم` بسعر **224.25** ريال — لكن المحرك لم يجده.

## السبب الجذري

المحرك لا يفهم أن "جدار" و"حوائط" نفس المفهوم. البند يقول **"جدار بسمك 150 مم"** والمكتبة تقول **"حوائط سمك 150مم"** — بدون ربط مرادفات بينهما، المحرك لا يراهما متشابهين.

كذلك لا يوجد Anti-Confusion يمنع خلط "جدار بلوك" مع "بلاطة مجوفة" أو "بلاط أرضي".

## الحل — 3 إجراءات

### 1. إضافة مرادفات "جدار/حوائط/حائط" — `synonyms.ts`

```text
إضافة مجموعة مرادفات جديدة:
"جدار_بلوك": ["جدار", "حوائط", "حائط", "wall", "جدران", "حيطان"]

+ مرادفات تفريق:
"بلاطة_مجوفة": ["بلاطة مجوفة", "hollow core slab", "بلاطه مجوفه", "هولوكور"]
"بلاط_ارضي" — موجود بالفعل
```

### 2. إضافة أزواج Anti-Confusion — `synonyms.ts`

```text
["جدار_بلوك", "بلاطة_مجوفة"]  — منع خلط جدار مع بلاطة سقف
["جدار_بلوك", "بلاط_ارضي"]   — منع خلط جدار مع بلاط أرضي
["جدار_بلوك", "بورسلان"]     — منع خلط جدار مع بورسلان
["بلاطة_مجوفة", "بلوك"]      — منع خلط بلاطة سقف مع بلوك
```

### 3. إعادة تعيين البنود المتضررة — Database

إعادة البندين (وأي بنود مشابهة) لحالة `pending`:
- جدار 150مم المربوط ببلاطة مجوفة
- جدار 200مم المربوط ببلاط تراتزو

## الملفات المتأثرة

| الملف | التغيير |
|---|---|
| `src/lib/pricing/synonyms.ts` | إضافة مرادفات جدار/حوائط + بلاطة مجوفة + أزواج Anti-Confusion |
| Database | إعادة تعيين البنود المتضررة لـ pending |

## النتيجة المتوقعة
- عند إعادة التسعير: جدار 150مم → يطابق "حوائط سمك 150مم" بسعر **189.75** ريال
- جدار 200مم → يطابق "حوائط سمك 200مم" بسعر **224.25** ريال
- بلاطة مجوفة وبلاط تراتزو لن يتداخلا مع بنود الجدران أبداً
