

# حذف مكتبة الأسعار + إصلاح موقع البحث

## 1. حذف بيانات مكتبة الأسعار

المكتبة حالياً تحتوي على **656 بند** + **39 سجل في rate_sources**.

سيتم حذف البيانات بالترتيب التالي (للحفاظ على سلامة البيانات):
1. حذف جميع سجلات `rate_sources` (المرتبطة بالمكتبة)
2. حذف جميع سجلات `rate_library`
3. إلغاء ربط البنود في `boq_items` بالمكتبة (`linked_rate_id = NULL`)

> **ملاحظة**: هذا لا يحذف جداول الكميات أو أسعارها — فقط يفصل الربط مع المكتبة القديمة.

بعد الحذف، يمكنك رفع ملف مكتبة أسعار جديد من خلال زر "رفع ملف أسعار" الموجود في الصفحة.

## 2. إصلاح موقع مربع البحث

مربع البحث حالياً تحت العنوان — سيتم نقله ليكون **في نفس سطر العنوان** على اليسار بعرض كامل متاح، مع الأزرار تحته.

```text
┌──────────────────────────────────────────────────┐
│  [🔍 بحث .............................] عنوان الصفحة │  ← سطر واحد
│  [تنزيل] [رفع ملف] [إضافة]              الوصف   │
│  [فئة1] [فئة2] [فئة3] ...                        │
│  ┌─────────────────────────────────────────────┐  │
│  │ الجدول                                      │  │
│  └─────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────┘
```
