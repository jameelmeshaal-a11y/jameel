

# إضافة إعادة تسعير بند محدد + تقرير ملخص المشروع

## 1. إعادة تسعير بند محدد (Per-Item Reprice)

### الموقع
في جدول الكميات (`BoQTable.tsx`)، عند تمرير الماوس على أي صف مسعّر، يظهر زر ⟲ بجانب زر العين 👁 الموجود حالياً.

### السلوك
- الضغط على الزر يعيد تسعير **هذا البند فقط** عبر استدعاء `runPricingEngine` المعدّل
- إضافة دالة `repriceSingleItem()` في `pricingEngine.ts` تأخذ `itemId` + `boqFileId` وتشغّل نفس pipeline المطابقة والتسعير على بند واحد
- تحديث الصف في الواجهة فوراً عبر `qc.setQueryData`
- إظهار toast بنتيجة التسعير (نجح / لم يُطابق / خطأ)

### الملفات
| الملف | التغيير |
|---|---|
| `src/lib/pricingEngine.ts` | إضافة `repriceSingleItem(boqFileId, itemId, cities, projectType)` |
| `src/components/BoQTable.tsx` | إضافة زر ⟲ لكل صف + handler |

## 2. تقرير ملخص جداول الكميات للمشروع

### الموقع
تبويب جديد **"ملخص"** في صفحة المشروع (`ProjectDetail.tsx`)، أو قسم ملخص يظهر فوق قائمة جداول الكميات في تبويب "جداول الكميات".

### المحتوى
جدول بسيط يعرض:

```text
┌──────────────────────┬───────────┬──────────────┬────────────┐
│ جدول الكميات         │ عدد البنود │ البنود المسعّرة │ الإجمالي    │
├──────────────────────┼───────────┼──────────────┼────────────┤
│ أعمال الكهرباء       │ 120       │ 95           │ 1,200,000  │
│ أعمال التكييف        │ 85        │ 85           │ 3,500,000  │
│ ...                  │ ...       │ ...          │ ...        │
├──────────────────────┼───────────┼──────────────┼────────────┤
│ الإجمالي             │ 205       │ 180          │ 4,700,000  │
└──────────────────────┴───────────┴──────────────┴────────────┘
```

### التنفيذ
- مكوّن جديد `ProjectBoQSummary` يستقبل `projectId`
- يجلب جميع `boq_files` + يحسب `boq_items` aggregates لكل ملف
- يعرض جدول ملخص + صف إجمالي
- يظهر فوق قائمة BoQ files في تبويب "جداول الكميات"

### الملفات
| الملف | التغيير |
|---|---|
| `src/components/ProjectBoQSummary.tsx` | مكوّن جديد — جدول ملخص |
| `src/pages/ProjectDetail.tsx` | إدراج المكوّن فوق قائمة الملفات |
| `src/lib/pricingEngine.ts` | إضافة `repriceSingleItem()` |
| `src/components/BoQTable.tsx` | إضافة زر إعادة تسعير بند محدد |

لا migrations. لا تأثير على المنطق الحالي.
